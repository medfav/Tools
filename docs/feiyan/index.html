抗流感药物奥司他韦、利巴韦林能预防新型冠状病毒
奥司他韦对于新型冠状病毒的增殖和释放都是无能为力的，药理学和临床上也缺乏相应的研究，因此不建议大家尝试这类方法来预防新型冠状病毒，也不建议常规使用奥司他韦预防流感。
戴多层口罩，才能有效预防新型冠状病毒
就医用口罩而言，只要正确佩戴合格产品，只需一个就能达到预期的防护效果。多个叠戴虽然理论上是能增加一些防护效果，但会增加通气阻力和佩戴的不适感，可能得不偿失。

普通人出门要佩戴护目镜，防止新型冠状病毒侵入体内
护目镜能挡下对着眼睛直接飞过来的那一部分飞沫，但是患者咳嗽时咳出的飞沫是放射状的，仍然有大量的微粒悬浮在空气中。真正密切接触患者的医务人员都应当佩戴泳镜式的专用防护眼镜，而普通人在一般生活场景中只需戴口罩就可以了。

钟南山院士建议盐水漱口防病毒
钟南山院士并未给出盐水漱口防病毒的建议，广州医科大学附属第一医院已辟谣。新型冠状病毒主要通过呼吸道粘膜侵入人体，而漱口清洁的是消化道，再加上冠状病毒并不具有完整细胞结构，盐水的渗透压杀菌原理对它无效，所以盐水漱口并不能防病毒。

新型肺炎预防须知

小知识

1. 传染源: 野生动物，可能为中华菊头蝠。

2. 病毒: 新型冠状病毒 2019-nCoV

3. 传播途径: 经呼吸道飞沫传播，亦可通过接触传播

4. 易感人群: 人群普遍易感。老年人及有基础疾病者感染后病情较重，儿童及婴幼儿也有发病

5. 潜伏期: 一般为 3~7 天，最长不超过 14 天，潜伏期内存在传染性
